# 기능 명세서 V2.0.1

# 기능 명세서 (Feature Spec)

> 타겟(페르소나): 투자자 60~70% / 일반 유저 20~30%
> 
> 
> **제약사항(필수 상기)**
> 
> - **현재(실시간) 데이터 사용 불가**: 모든 데이터는 과거 데이터 기반
> - **투자 판단/추천 금지**: 우리는 **데이터 시각화/정보 제공**, 판단은 사용자에게 맡김
> - **수집 불가/구현 불가 기능은 과감히 제외 가능**

# 화면 구조 제안 (Navbar)

- **홈(대시보드)**: 핵심 지표 카드 + 요약 차트 + 최근 본 아파트 + 뉴스 헤드라인
- **지도**: 검색/필터 + “내 화면 아파트 가져오기” + 마커/상세 패널 + (선택) 로드뷰/히트맵
- **랭킹/통계**: 상승·하락·거래량·가격 랭킹 + 조건 필터
- **마이페이지(내 부동산)**: 즐겨찾기/최근기록/설정/언어

---

# 공통 UI/문구 규칙(필수)

- 모든 지표/차트 화면에 고정 문구:
    - “본 서비스는 **과거 데이터 기반 시각화**이며 **투자 판단/권유를 제공하지 않습니다.**”
- 데이터 출처/기간 표기(가능 시):
    - “데이터 기준 기간: YYYY-MM ~ YYYY-MM” / “출처: OOO”
